### 🚀 **How to Use**

1. **Launch the Tool**:
   ```bash
   cd /Users/admin/Documents/faceswap/faceswap
   ./launch_zero_config.sh
   ```

2. **Open Web Interface**: 
   - Automatically opens at http://localhost:7860
   - Or click the Simple Browser link in VS Code

3. **Upload Images**:
   - **Source Face**: Upload an image with the face you want to use
   - **Target Image**: Upload an image where you want to replace the face
   - **Face Index**: Select which face to replace (0 for first face)

4. **Get Results**: Click submit and get your face-swapped image instantly!
